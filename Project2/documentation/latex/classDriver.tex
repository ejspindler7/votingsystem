The constructor for the \mbox{\hyperlink{classDriver}{Driver}} class. 

This function will be used to create a new driver to process a C\+SV file with the given filename.

The file is assumed to be formatted correctly, so there is no exception handling for incorrect C\+SV files. If the C\+SV file is not arranged as expected, this can lead to issues with program function, but it is considered improper use of the system and should not be expected to execute.
